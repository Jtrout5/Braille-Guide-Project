This is a small project authored in whole by Jeramiah Trout (me) with assitance from Selene Monjaraz.

The code is my original work, and her assistance comes in the confirmation of brsille lettering, notation, and accuracy

The primary goal of this project is to allow users to write text or input files with text (in either pdf, docx, or txt format, of which txt will likely work the best) and be given the guide to type the given text as braille on a brailler.

Please note that this is an incredibly niche use case, and might never be used outside of my own personal use. I intend to have it visually display what keys to use on a 6 dot brailler, and allow options for manually advancing or automatically advancing with user directed speeds, rewinding, and pausing as well

As of 03/22/2026, the project simply generates a json file containing all* of the currently accepted UEB letters, digits, contractions, wordsigns, groupsigns, etc and prints them to the terminal.

Just a proof of concept that this can work at the moment

* I removed some of the unneccessary UEB cntractions which Selene and I agreed were unneded for this project since the goal is for me to be able to type notes in standard print english, input them and then type in braille. Words I would never use were removed. They can be added later if this project expands past my solo use case.
